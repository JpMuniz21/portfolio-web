# 💻 Portfólio Pessoal - João Paulo (Jota)

Este projeto foi desenvolvido como requisito para a disciplina de **Desenvolvimento de Plataformas Web** na **UNIFOR**. O foco principal foi a criação de uma interface de alto impacto visual utilizando apenas tecnologias fundamentais de front-end.

## 🎨 Design & UI/UX
- **Tema Dracula**: Paleta de cores focada em produtividade e estética developer.
- **Glassmorphism**: Tabelas e seções com efeitos de transparência e desfoque de fundo (`backdrop-filter`).
- **Tipografia**: Uso da fonte **JetBrains Mono** para reforçar a identidade de programação.
- **Interatividade 100% CSS**: Efeito de digitação (*typing effect*) realista e animações de entrada (*fade-in*) controladas puramente por CSS.

## 🛠️ Tecnologias Utilizadas
- **HTML5 Semântico**: Estrutura organizada para acessibilidade e SEO.
- **CSS3 Avançado**: Uso intenso de `@keyframes`, Flexbox, Gradients e filtros de brilho (*drop-shadow*).
